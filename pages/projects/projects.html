Os meus Projetos

Projeto MEDISYNC - R&D CAPGEMINI

Na empresa onde estou atualmente integro um projeto com uma dimensão incrível. Este projeto consiste em criar uma plataforma de gestão de pacientes de um hospital, através do desenvolvimento de uma interface interativa, rápida e segura, com inclusão de estatísticas dinâmicas, informações sobre os médicos e pacientes associados. O objetivo é fazer algo que sirva os utilizadores de um modo mais eficiente para responder à urgência nos pedidos de resposta dentro do hospital. Este projeto tem muito potencial uma vez que surgiu de uma sondagem do público-alvo, em que a maioria dos interessados indicaram falhas na responsividade e atualização dos sistemas já existentes.

Competências: SQL | HTML & CSS | Node js | JavaScript | React js | Python | C

Projeto BIOINTEGRATION - Universidade do Minho

Neste projeto em parceria com a Universidade do Minho integrei a equipa de front-end para a criação das páginas de um website para recursos de bioinformática. O objetivo deste projeto é oferecer um ambiente educacional o mais cativante possível, com conteúdos bem organizados por níveis de dificuldade e estruturados por categoria. Características do website:

  • Design responsivo com interface moderna para um experiência imersiva;
  • Layout dinâmico com integração de elementos visuais apelativos em formato vídeo e imagem.

Competências: MongoDB | HTML & CSS | SASS | Node js | JavaScript | Laravel
